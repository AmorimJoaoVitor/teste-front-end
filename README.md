# Teste Front-End Econverse

Projeto desenvolvido em React + TypeScript + SCSS conforme layout proposto no teste técnico.

## Tecnologias utilizadas

* React
* TypeScript
* SCSS
* Embla Carousel
* Lucide Icons

## Como rodar o projeto

### Instalar dependências

npm install

### Rodar ambiente local

npm run dev

## Build de produção

npm run build

## Funcionalidades

* Header responsivo
* Banner principal
* Categorias interativas
* Vitrine de produtos consumindo API JSON
* Modal de produto
* Newsletter
* Footer responsivo

## API utilizada

https://app.econverse.com.br/teste-front-end/junior/tecnologia/lista-produtos/produtos.json
